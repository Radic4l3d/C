# 03_Estructuras_Datos - Estructuras de Datos

Este directorio contiene implementaciones de estructuras de datos fundamentales.

## Temas cubiertos:

- **Estructuras (struct)**: definición y uso
- **Listas enlazadas**:
  - Simples
  - Dobles
  - Circulares
- **Pilas (Stacks)**: LIFO (Last In, First Out)
- **Colas (Queues)**: FIFO (First In, First Out)
- **Árboles**:
  - Árboles binarios
  - Árboles binarios de búsqueda (BST)
  - Recorridos: inorden, preorden, postorden
- **Grafos**: representación y operaciones básicas
- **Tablas hash**: implementación básica

## Estructura sugerida:

```
03_Estructuras_Datos/
├── listas_enlazadas.c      # Listas simples y dobles
├── pilas.c                 # Implementación de pilas
├── colas.c                 # Implementación de colas
├── arboles_binarios.c      # Árboles y BST
├── grafos.c                # Representación de grafos
└── tablas_hash.c           # Hash tables
```

## Compilación:

```bash
gcc -Wall -Wextra nombre_archivo.c -o nombre_programa
./nombre_programa
```
