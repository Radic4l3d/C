# 01_Basicos - Fundamentos de C

Este directorio contiene ejercicios y ejemplos sobre los fundamentos básicos de C.

## Temas cubiertos:

- [x] **Variables y tipos de datos**: int, float, char, double
- [ ] **Operadores**: aritméticos, lógicos, relacionales
- [ ] **Entrada/salida**: printf(), scanf()
- [ ] **Estructuras de control**:
  - [ ] Condicionales: if, else, switch
  - [ ] Ciclos: for, while, do-while
- [ ] **Funciones básicas**

## Estructura sugerida:
```
01_Basicos/
├── variables.c         # Declaración y uso de variables
├── operadores.c        # Operadores aritméticos y lógicos
├── condicionales.c     # if, else, switch
├── ciclos.c            # for, while, do-while
└── funciones.c         # Definición y uso de funciones
```

## Compilación:

```bash
gcc nombre_archivo.c -o nombre_programa
./nombre_programa
```
